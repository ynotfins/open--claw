# TOOLS INVENTORY - Complete Arsenal ⚡

## Core OpenClaw Tools
- **read**: Read file contents (text/images)
- **write**: Create or overwrite files  
- **edit**: Make precise edits to files
- **exec**: Run shell commands (pty available for TTY-required CLIs)
- **process**: Manage background exec sessions
- **browser**: Control web browser (Chrome/isolated)
- **canvas**: Present/eval/snapshot the Canvas
- **nodes**: List/describe/notify/camera/screen on paired nodes
- **cron**: Manage cron jobs and wake events (reminders)
- **message**: Send messages and channel actions
- **gateway**: Restart, apply config, or run updates
- **tts**: Convert text to speech

## Session Management
- **agents_list**: List OpenClaw agent ids for subagent spawning
- **sessions_list**: List other sessions with filters
- **sessions_history**: Fetch history for another session
- **sessions_send**: Send message to another session
- **subagents**: List, steer, or kill sub-agent runs
- **session_status**: Show usage/time/cost status
- **sessions_spawn**: Spawn isolated sub-agent or ACP coding session

## Web & Content
- **web_search**: Search web using Perplexity Sonar
- **web_fetch**: Fetch and extract readable content from URLs
- **image**: Analyze images with vision model
- **pdf**: Analyze PDF documents

## Memory & Persistence
- **memory_search**: Semantically search MEMORY.md + memory/*.md
- **memory_get**: Safe snippet read from memory files
- **whatsapp_login**: Generate WhatsApp QR code for linking

- **self-improvement**: Capture learnings and corrections
